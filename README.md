# projekt3
Landing Page
Development Institute - Landing Page
Dieses Projekt ist eine moderne und responsive Landingpage für ein fiktives Bildungsinstitut, das Kurse im Bereich Softwareentwicklung anbietet. Es demonstriert meine Fähigkeiten in der Erstellung benutzerfreundlicher Interfaces mit HTML5, CSS3 und JavaScript.

Features
Interaktiver Header: Enthält ein funktionales Dropdown-Menü für die Navigation sowie ein einkaufswagen-Icon für eine intuitive User Experience.

Kurs-Katalog (Main Menu): Eine strukturierte Übersicht der angebotenen Entwicklungskurse mit detaillierten Informationen.

Responsive Design: Die Seite ist für verschiedene Bildschirmgrößen optimiert.

Social Connectivity: Ein strukturierter Footer mit Verknüpfungen zu sozialen Medien zur besseren Erreichbarkeit.

Technologien
Frontend: HTML5, CSS3 

Logik: JavaScript 

Versionierung:  GitHub

Struktur
index.html - Struktur der Landingpage

style.css - Design und Responsive Layouts

script.js - Interaktive Elemente
